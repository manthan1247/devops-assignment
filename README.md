# devops-assignment

## Project Overview
This project demonstrates a complete CI/CD pipeline using GitHub Actions and AWS EC2.

## Technologies Used
- AWS EC2
- Ubuntu
- Nginx
- Git
- GitHub
- GitHub Actions (CI/CD)

## Project Architecture

GitHub Repository
        ↓
GitHub Actions
        ↓
AWS EC2 (Ubuntu)
        ↓
Nginx
        ↓
Website

## Deployment Steps

1. Created AWS EC2 Instance.
2. Installed Nginx.
3. Created GitHub Repository.
4. Cloned Repository on EC2.
5. Added Website Files.
6. Configured GitHub Actions.
7. Automatic Deployment using CI/CD.

## Website

The website is hosted on AWS EC2 using Nginx.
